## 坦克大战
#### 项目简介：
就是根据按照韩顺平老师视频的指导写的一个小项目，巩固一下java se阶段的学习，该项目并不完善。
#### 功能介绍：
1. 可以控制坦克移动和攻击
2. 有敌方坦克，并会移动和攻击
3. 有音效和爆炸效果
4. 可以保存游戏，并可以接着上一盘玩
#### 运用知识：
JAVA SE 的部分内容
#### 开发环境：
Eclipse
#### 使用方法：
1. `git clone https://github.com/codingXiaxw/CustomerManagement.git`
2. 使用Eclipse导入打开
